# Fancy Text Studio

Unicode text transformer for bios, usernames, and social profiles.

---

## Features

- **30+ styles** — Gothic, Double Struck, Cursive, Serif Bold/Italic, Sans, Bubble, Vaporwave, and more
- **Distortion effects** — Zalgo glitch, strikethrough, slash, dotted, upside-down, ghostly
- **Preset frames** — sword, crown, fire, diamond, music, heart wings, anime brackets
- **Style Mixer** — blend two styles into one hybrid result
- **Decor Builder** — custom prefix/suffix wrapper with live preview
- **Case tools** — uppercase, lowercase, title, alternate, inverse, reverse, mirror
- **Text tools** — add/remove spacing, leet speak, lorem ipsum, star/heart wrapping
- **Undo / Redo** — full edit history stack
- **Social meters** — live character limit bars for Twitter/X (280), Instagram (150), TikTok bio (80)
- **Text stats** — character frequency chart, letter/digit/space breakdown, estimated read time
- **Social preview panel** — see how text looks in Twitter, Instagram, and Discord bios
- **Input history** — auto-saves recent inputs, click to restore
- **Favorites** — star any style, filter to favorites only
- **4 themes** — Dark, Warm, Neon, Light
- **Export** — copy all styles or download as `.txt`

## Usage

Open `achickonweb.github.io/FancyTextStudio` in any modern browser. No build step, no dependencies, no server required.


## Stack

Vanilla HTML, CSS, JavaScript. Zero frameworks. Zero external requests (fonts load from Google Fonts).
